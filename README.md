# Waller AI

Angular frontend project built with Tailwind CSS.

## Tech Stack
- Angular
- TypeScript
- Tailwind CSS

## Run Locally
npm install
npm start

## Local OAuth Dev Note
- Before starting Angular, make sure port `4200` is free.
- Run the dev server with `ng serve --host localhost --port 4200 --proxy-config proxy.conf.json` or just `npm start`.
- Local OAuth testing must use `http://localhost:4200`.
- Never accept Angular's random alternative port prompt for Google OAuth testing.
